# Landing Zone Propeller

An open-source framework for AWS multi-account landing zones. Deploy and manage
your organization's infrastructure with a standardized, repeatable approach.

Propeller provides a pipeline engine that coordinates resource deployments
across accounts using standard tools like Terraform, handles cross-project
dependencies, and supports customization through overlays and overrides. No need
to fork the framework.

It ships with a customizable, best-practice landing zone out of the box.
