# Examples

Reference projects and a tiny example pipeline. Use these to:

- See a minimal valid project layout for each deploy type.
- Copy as a starting point for new projects.
- Read alongside [`docs/project-structure.md`](../project-structure.md) and
  [`docs/customization.md`](../customization.md).

## What's here

- `propeller.yaml` - a two-step pipeline that demonstrates cross-project
  input/output wiring.
- `projects/example-terraform/` - minimal Terraform project.
- `projects/example-script/` - minimal script (just-based) project.
